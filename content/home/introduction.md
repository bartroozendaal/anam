# Welcome to Anam

This is a beautiful, content-driven website built with Angular.

## Key Features

- **Markdown Content**: Write your content in markdown and it will be automatically rendered
- **Responsive Grid**: 16-column responsive grid system that adapts to all screen sizes
- **Simple Navigation**: Clean header with logo and navigation links
- **Footer Links**: Quick access to important pages from the footer

## Getting Started

Explore the site using the navigation menu at the top of the page.
